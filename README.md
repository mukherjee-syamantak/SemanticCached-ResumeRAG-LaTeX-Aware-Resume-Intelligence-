# SemanticCached-ResumeRAG-LaTeX-Aware-Resume-Intelligence-
Standard RAG drops LaTeX resume hyperlinks and runs costly redundant queries. This free, local Resume RAG system uses Spatial Coordinate Mapping to bind hidden URLs to visible text for 100% link retention. A Dual-Layer Semantic Cache deduplicates paraphrased questions, serving answers with zero latency via Hugging Face models.
